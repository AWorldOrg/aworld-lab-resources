# @aworld/app-sdk

TypeScript SDK for the AWorld Consumer API (App).

> **Coming soon.** This SDK will be auto-generated from [`schemas/app.openapi.json`](../../schemas/app.openapi.json) and published via the Bitbucket CI pipeline.

In the meantime, you can use the OpenAPI schema directly to generate a client with your preferred tooling:

```bash
bunx openapi-typescript ../../schemas/app.openapi.json -o app-schema.d.ts
```

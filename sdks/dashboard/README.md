# @aworld/dashboard-sdk

TypeScript SDK for the AWorld Admin API (Dashboard).

> **Coming soon.** This SDK will be auto-generated from [`schemas/dashboard.openapi.json`](../../schemas/dashboard.openapi.json) and published via the Bitbucket CI pipeline.

In the meantime, you can use the OpenAPI schema directly to generate a client with your preferred tooling:

```bash
bunx openapi-typescript ../../schemas/dashboard.openapi.json -o dashboard-schema.d.ts
```
